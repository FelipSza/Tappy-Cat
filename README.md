# Tappy-Cat 🐱🎮

> Meu primeiro jogo, inspirado no clássico Flappy Bird!

Um projeto divertido e simples focado em recriar a mecânica viciante de desviar de obstáculos com apenas um toque, desenvolvido como uma forma de aprendizado e introdução ao desenvolvimento de jogos.

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** [C#](https://docs.microsoft.com/pt-br/dotnet/csharp/)
* **Game Engine:** [Unity](https://unity.com/)
  
## 🚀 Como Executar o Projeto

Se você deseja baixar e testar o jogo na sua máquina, siga os passos abaixo:

### Pré-requisitos
* Ter o **Unity Hub** e a versão correspondente do **Unity Editor** instalada.
* Ter o **Git** instalado.
